# CM3045 3D Week 8 Animation Exercise

A Unity 3D animation exercise for CM3045, focused on an animated sci-fi drone scene.

## Overview

This project builds a small interactive drone demo using Unity animation states, keyboard input, particle effects, lighting, and a simple HUD.

## What I Built

- Animated drone state machine
- Launch, dock, flight, scan, and emergency modes
- Keyboard-controlled drone movement
- Thruster particle effects
- Scan light behavior
- Follow camera
- On-screen HUD showing controls and current drone mode
- Sci-fi docking scene and WebGL build

## Tech Stack

- Unity
- C#
- Unity Input System
- Universal Render Pipeline
- WebGL build
